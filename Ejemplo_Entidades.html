Ejemplos de Entidades

Uso de Entidades HTML

Las entidades HTML se utilizan para representar caracteres especiales que tienen un significado particular en HTML. Estas entidades comienzan con un ampersand (&) y terminan con un punto y coma (;).

Ejemplo 1: Símbolo de menor que

Para mostrar el símbolo de menor que (<), puedes usar la entidad: &lt;

Ejemplo 2: Símbolo de mayor que

Para mostrar el símbolo de mayor que (>), puedes usar la entidad: &gt;

Ejemplo 3: Símbolo de ampersand

Para mostrar el símbolo de ampersand (&), puedes usar la entidad: &amp;

Ejemplo 4: Comillas dobles

Para mostrar comillas dobles ("), puedes usar la entidad: &quot;

Ejemplo 5: Comillas simples

Para mostrar comillas simples ('), puedes usar la entidad: &#39;
