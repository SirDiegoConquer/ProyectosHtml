Ejemplos de code

Uso del elemento <code>

El elemento <code> se utiliza para definir fragmentos de código en línea dentro de un texto. Es útil para resaltar comandos, funciones o cualquier otro tipo de código.

Ejemplo 1: Código HTML

Para crear un enlace en HTML, puedes usar la siguiente sintaxis: <a href="URL">Texto del enlace</a>

Ejemplo 2: Código JavaScript

Para mostrar un mensaje en la consola, puedes usar el siguiente comando: console.log("Hola, mundo!");

Ejemplo 3: Código CSS

Para cambiar el color de fondo de una página, puedes usar la siguiente regla CSS: body { background-color: lightblue; }
